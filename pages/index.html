About Myself

I go by "Average Helper" online, partly because my name is super generic. An average name, really. I've been doing software things since 2011. I try to be helpful where I can, especially to write respectful software.

Stuff I do for fun

Here's just a few things I worked on myself that I'm proud of. (You can check my GitHub or Codeberg for more.)

  • Gamgee – A fun little Discord bot to help manage a music queue for public DJ events. Written in TypeScript.
  • meta-comments – The first VS Code plugin I ever published! It decorates code based on specially-formatted comments.
  • ItemEncryptor – A Swift package to make data encryption easy. I use this in some of my smaller projects to facilitate end-to-end encryption of sensitive data.
  • Recorded Finance – An end-to-end encrypted finance app for keeping track of account balances. I had a version of this written for iOS in Swift, but I've began to port that project to the web for better cross-platform compatibility and accessibility. The project is still in its infant stages, with several front-end glitches I want to fix before I "release" the project into the wild. The front-end is written in Svelte, with TypeScript on the front- and back-end.
  • Trump Donald – An iOS application based on the game Whac-a-mole.
  • My blog – An experiment to see how GitHub Pages work. I post ramblings there from time to time. The posts are written in Markdown, and the site is generated automagically using Zola.

Stuff I get paid for

My personal work is mainly with open-source projects and mobile applications. Since starting at Inventory Shield, LLC in 2018, I began working with web technologies. Inventory Shield's tech stack is TypeScript and Vue.js on the front-end, and Node.js (of course with TypeScript) on the back-end. The products are web apps, and we use Apache Cordova™ to wrap them in iOS and Android app shells for better device features.

Unfortunately, the company's code is closed-source. You can check out their inventory management and referral management products at their respective sites.
